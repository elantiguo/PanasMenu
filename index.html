Welcome to the Minecraft Friendship Revolution!

¡Bienvenido a la Revolución de la Amistad en Minecraft!

Overview / Descripción

Are you tired of playing alone on your Minecraft server? Want to manage your friendships in a fun and simple way? Look no further! PanasMenuFree is the plugin you need to transform your gaming experience. With this plugin, you can add friends, block unwanted players, and best of all, send gifts to your adventure companions!

¿Estás cansado de jugar solo en tu servidor de Minecraft? ¿Quieres gestionar tus amistades de una manera divertida y sencilla? ¡No busques más! PanasMenuFree es el plugin que necesitas para transformar tu experiencia de juego. Con este plugin, podrás agregar amigos, bloquear a quienes no deseas tener cerca y, lo mejor de todo, ¡enviar regalos a tus compañeros de aventuras!

Key Features / Características Destacadas

Friend Management

Add and remove friends with just a few clicks. Having friends close by has never been easier!

Gestión de Amigos

Agrega y elimina amigos con un par de clics. Nunca ha sido tan fácil tener a tus amigos cerca.

Player Blocking

Is a player bothering you? Block them! No more unwanted interruptions.

Bloqueo de Jugadores

¿Un jugador te está molestando? ¡Bloquealo! No más interrupciones no deseadas.

Gift Giving

Surprise your friends by sending items directly from your inventory. A gesture that will always be appreciated!

Envío de Regalos

Sorprende a tus amigos enviándoles ítems directamente desde tu inventario. ¡Un gesto que siempre será apreciado!

Real-time Notifications

Receive alerts when your friends connect or disconnect. Stay up to date with their adventures!

Notificaciones en Tiempo Real

Recibe alertas cuando tus amigos se conectan o desconectan. ¡Siempre estarás al tanto de sus aventuras!

Magic Commands / Comandos Mágicos

  • /panas - Open the main friends menu / Abre el menú principal de gestión de amigos
  • /helppanas - Display help information / Muestra información de ayuda
  • /aceptar <player> - Accept a friend request / Acepta una solicitud de amistad
  • /rechazar <player> - Reject a friend request / Rechaza una solicitud de amistad
  • /panasreload - Reload plugin configuration / Recarga la configuración del plugin

Installation / Instalación

  1. Download the PanasMenuFree .jar file / Descarga el archivo .jar de PanasMenuFree
  2. Place the file in your server's plugins folder / Coloca el archivo en la carpeta de plugins de tu servidor
  3. Restart your server / Reinicia tu servidor
  4. Configure the plugin through config.yml / Configura el plugin a través de config.yml
  5. Verify the plugin is working with /panas command / Verifica que el plugin esté funcionando con el comando /panas

How to Use / Cómo Usar

  1. Start the Adventure: Use /panas to open the main menu
  2. Add Friends: Select "Add friend" and type the player's name
  3. Remove Friends: Choose "Remove friend" and follow the instructions
  4. Block & Unblock: Use blocking options to maintain your personal space
  5. Send Gifts: Hold the item you want to send, select "Send Gift" and specify the recipient
  1. ¡Comienza la Aventura!: Usa el comando /panas para abrir el menú principal
  2. Agrega Amigos: Selecciona "Añadir amigo" y escribe el nombre del jugador
  3. Elimina Amigos: Si necesitas eliminar un amigo, selecciona "Eliminar amigo"
  4. Bloquea y Desbloquea: Usa las opciones de bloqueo para mantener tu espacio personal
  5. Envía Regalos: Sostén el ítem, selecciona "Enviar Regalo" y especifica el destinatario
