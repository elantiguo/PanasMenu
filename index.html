PanasMenu

Transform your Minecraft server's social experience

Transforma la experiencia social de tu servidor Minecraft

Overview / Descripción

PanasMenu is an innovative plugin designed for Minecraft servers that enables players to manage their friendships effortlessly through an intuitive interface. With features like friend requests, gift-giving, and real-time notifications, PanasMenu enhances the social experience of your Minecraft server.

PanasMenu es un plugin innovador diseñado para servidores de Minecraft que permite a los jugadores gestionar sus amistades de manera sencilla a través de una interfaz intuitiva. Con características como solicitudes de amistad, envío de regalos y notificaciones en tiempo real, PanasMenu mejora la experiencia social de tu servidor de Minecraft.

Key Features / Características Principales

Interactive Friends Menu

User-friendly visual interface for managing friends without complex commands

Menú de Amigos Interactivo

Interfaz visual amigable para gestionar amigos sin comandos complejos

Friend Management

Add, remove, block, and unblock friends with ease

Gestión de Amigos

Añade, elimina, bloquea y desbloquea amigos fácilmente

Gift System

Send in-game items to friends directly through the interface

Sistema de Regalos

Envía objetos del juego a tus amigos directamente a través de la interfaz

Real-time Notifications

Get notified when friends connect, disconnect, or send requests

Notificaciones en Tiempo Real

Recibe notificaciones cuando tus amigos se conectan, desconectan o envían solicitudes

Commands / Comandos

  • /friends - Open the friends management menu / Abrir el menú de gestión de amigos
  • /helpfriends - Display available commands and their usage / Mostrar comandos disponibles y su uso
  • /accept <player> - Accept a friend request / Aceptar una solicitud de amistad
  • /reject <player> - Reject a friend request / Rechazar una solicitud de amistad

Installation / Instalación

  1. Download the PanasMenu .jar file / Descarga el archivo .jar de PanasMenu
  2. Place the file in your server's plugins folder / Coloca el archivo en la carpeta de plugins de tu servidor
  3. Restart your server / Reinicia tu servidor
  4. Configure the plugin through config.yml / Configura el plugin a través de config.yml
  5. Verify the plugin is working with /friends command / Verifica que el plugin esté funcionando con el comando /friends

Configuration / Configuración

menu:
      # Configuración del menú de amigos
      menu:
        add_friend:
          material: "DIAMOND"  # Material del ítem para "Agregar Amigo"
          name: "Add Friend"     # Nombre del ítem
          lore: "Click to add a friend"  # Lore del ítem
        remove_friend:
          material: "REDSTONE"   # Material del ítem para "Eliminar Amigo"
          name: "Remove Friend"
          lore: "Click to remove a friend"
        block_friend:
          material: "BARRIER"     # Material del ítem para "Bloquear Amigo"
          name: "Block Friend"
          lore: "Click to block a friend"
        unblock_friend:
          material: "GREEN_WOOL"  # Material del ítem para "Desbloquear Amigo"
          name: "Unblock Friend"
          lore: "Click to unblock a friend"
        send_gift:
          material: "PAPER"        # Material del ítem para "Enviar Regalo"
          name: "Send Gift"
          lore: "Click to send a gift"
        friends_list:
          material: "BOOK"        # Material del ítem para "Lista de Amigos"
          name: "Friends List"
          lore: "Click to view your friends"
      
      # Mensajes utilizados en el plugin
      messages:
        has_connected: "%player% has connected."
        has_disconnected: "%player% has disconnected."
        player_not_online: "The player is not online."
        friend_not_found: "Friend %friend% not found."
        friend_already_added: "You are already friends with %friend%."
        friend_request_sent: "Friend request sent to %friend%."
        friend_request_received: "You have received a friend request from %player%."
        friend_request_accepted: "You have accepted the friend request from %friend%."
        friend_request_not_found: "No friend request found from %friend%."
        friend_request_rejected: "You have rejected the friend request from %friend%."
        friend_removed: "You have removed %friend% from your friends."
        friend_blocked: "You have blocked %friend%."
        friend_unblocked: "You have unblocked %friend%."
        friend_already_blocked: "You have already blocked %friend%."
        gift_sent: "Gift sent to %friend%."
        gift_received: "You have received a gift from %player%."
